WebNovel Summarizer

AI-Powered Reading Assistant for Web Novel Enthusiasts

Elevator Pitch

A sophisticated Android app combining a smart web browser with AI-powered chapter summarization for web novel readers. Supports 8+ popular novel sites, offers three summary depth levels, and stores your entire library offline — all without an account or cloud sync.

Key Features

Smart WebView Browser
AI Summarization
3-Level Summaries
Novel Library
Browsing History
Bookmarks
Text-to-Speech
Offline Access

Tech Stack

Kotlin MVVM Room Database Groq AI Cerebras AI Retrofit Coroutines WebView
Implementation Highlight: Built a multi-layer JavaScript content extraction engine that removes ads and navigation elements, tries multiple CSS selectors with a fallback to paragraph filtering, and supports a manual selector override for tricky sites. Used a ContentHolder singleton to safely pass large novel text between activities, avoiding Android's "Transaction too large" crash.

Future Improvements

  • Batch summarization across multiple chapters
  • Export library to PDF/EPUB
  • Multi-model AI support (Claude, Gemini, etc.)
